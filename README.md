## Installation

```bash
npm install
```

## Running

```bash
node server.js
```

Then open [http://localhost:8000/](http://localhost:8000/)

## API

| Url        | Params           | Type |
| ------------- |:-------------:| :-----:| 
| /api/api/ocr | images | Files |
